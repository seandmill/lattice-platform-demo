# Lattice Platform Demo

## Demo Features

The Lattice Platform Demo provides an interactive, step-wise visualization of a request traversing through the Lattice architecture.

- Flow Canvas - Interactive visualization of the request flow through the Lattice architecture.
- Sequence Diagram - Step-by-step breakdown of the request flow.
- Payload Inspector - Detailed inspection of request and response payloads.
- Control Panel - Adjust request parameters and trigger actions.
- Breadcrumbs - Navigate through the request flow.
